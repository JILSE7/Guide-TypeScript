Interfaces

Una interfaz es muy similar a un type, realmente realizan lo mismo, queda mucho a discrecion de la personas, escoger entra cada una de ellas, la unica diferencia es que una interfaz puede ser extendida, añadiendio mas campos, es como si fueran heredadas

recuerda que las interfaces son implementadas

y extendidas para añadir mas campos

Interfaces

interface Animal { name: string } interface Bear extends Animal { honey: boolean } const bear = getBear() bear.name bear.honey

type

type Animal = { name: string } type Bear = Animal & { honey: boolean } const bear = getBear(); bear.name; bear.honey;
